# Curriculum Plugin

Component for managing learning plans, versions, and learning cycles.

## Requirements

- Moodle 4.5 or higher

## Installation

1. Place this plugin in `/local/curriculum/`
2. Visit Moodle administration site notifications page
3. The plugin will be automatically installed
